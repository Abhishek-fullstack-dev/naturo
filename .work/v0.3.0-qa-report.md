## 3.1 Features

### 3.1.1 Unified Selector: `naturo click "Button:1" --app Calculator` → PASS ✅ (with caveat)
- **Command**: `naturo click "Button:一" --app Calculator`
- **Result**: Successfully clicked (Calculator displayed "11" after two clicks)
- **Caveat**: On Chinese-locale Windows, button names are Chinese ("一" not "1"). The selector `"Button:1"` fails with "element not found". This is expected locale behavior but the release guide's example assumes English locale.
- **Note**: `naturo click "Button:Save"` syntax works correctly.

### 3.1.2b Backward compat: `naturo window focus "Notepad"` → FAIL ❌
- **Command**: `naturo window focus "Notepad"` (positional argument)
- **Expected**: Works with deprecation warning
- **Actual**: `Error: Got unexpected extra argument (Notepad)`
- **Workaround**: `naturo window focus --app Notepad` works
- **Issue**: `window focus` doesn't accept positional NAME arg like the old v0.2.1 syntax. Docs say `naturo window focus "Notepad"` should work.
- **Note**: `naturo window` subgroup exists and shows "deprecated — use 'naturo app' instead" but individual subcommands require `--app` flag
